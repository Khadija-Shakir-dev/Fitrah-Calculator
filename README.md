The Fitrah Calculator is a web-based tool designed to help Muslims quickly calculate their Zakat al-Fitr (Fitrah) before Eid-ul-Fitr. It provides accurate calculations based on the number of family members, the country-specific rates, and the type of staple food chosen (Wheat, Dates, Rice, Barley, Raisins, etc.).

This calculator ensures users can fulfill their religious obligations on time while taking into account local food prices and currencies across multiple countries. The interface is simple, user-friendly, and includes real-time validation to prevent errors.

Key Features:

Country-based rates for 35 countries.
Multiple food types for accurate calculation.
Responsive and intuitive UI with clear result display.
Guidance notes about who should pay, when, and why Fitrah is important.
🛠️ Technologies Used
HTML5 – For structure and markup.
CSS3 – For styling and responsive design.
JavaScript – For dynamic calculations and interactivity.
Optional: Google Fonts for better typography.
